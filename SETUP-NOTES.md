# Little Ommani Paperie update notes

## Product imagery
This version uses styled mockups only. Clean flat artwork and close-up files are not included in the website package.

## Cart setup
The cart will not function until you paste your Snipcart **public API key** into `snipcart-config.js`.

See `SNIPCART-SETUP.md` for the exact steps. The key is intentionally stored in one separate file so future website updates do not wipe it out.

## Uploading to GitHub
Upload all files inside this folder to the root of the `Little-Ommani-Paperie` repository. Replace matching files, but keep your completed `snipcart-config.js` when you receive future design-only updates.
